# Frontend - Muzic Spotify Clone

React.js application for music streaming user interface.

## Installation

```bash
cd frontend
npm install
cp .env.example .env
npm start
```

## Environment Variables

Create `.env` file:

```env
REACT_APP_API_URL=http://localhost:5000/api/v1
REACT_APP_CLOUDINARY_NAME=your_cloudinary_name
```
