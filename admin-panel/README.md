# Admin Panel - Muzic Spotify Clone

Professional Admin Dashboard for managing Muzic platform.

## Installation

```bash
cd admin-panel
npm install
cp .env.example .env
npm start
```

## Features

✅ Dashboard with Analytics
✅ Song Management
✅ User Management
✅ Playlist Management
✅ Revenue Tracking
✅ Auto Playlist Generation
✅ System Settings
✅ Backup & Export
